
## O que é o OWASP WSTG?

O OWASP Web Security Testing Guide (WSTG) é um **guia abrangente** que serve como metodologia para engagements de pentest de aplicações web.

Ele cobre todas as fases do teste, desde a coleta de informações até a exploração de vulnerabilidades, garantindo que nenhuma área importante seja esquecida durante o teste.



## Por que usar uma metodologia?

- Garante **cobertura completa** de todas as áreas de teste
- Fornece um **framework estruturado** para seguir passo a passo
- Ajuda a **documentar** o processo de forma organizada
- É reconhecido pela **indústria** como referência para pentests web
- Serve como **checklist** para não pular etapas críticas



## Categorias de Teste do WSTG

O guia organiza os testes em categorias como:

- Information Gathering
- Configuration and Deployment Management Testing
- Identity Management Testing
- Authentication Testing
- Authorization Testing
- Session Management Testing
- Input Validation Testing
- Error Handling Testing
- Cryptography Testing
- Business Logic Testing
- Client-Side Testing
